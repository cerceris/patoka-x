#### To use `patoka-x` in projects:

Execute in the project's folder.

    ```
    $ npm link <path-to-patoka-x>
    ```

Create link to use in runtime. Execute in `patoka-x/node_modules`.

    ```
    $ ln -s ../build patoka
    ```
